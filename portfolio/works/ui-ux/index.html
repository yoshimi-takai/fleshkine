ストックフォトサイト

PIXTA

企業名 | ピクスタ株式会社

URL | https://pixta.jp

ストックフォトサイトのリードデザイナーとしてデザイン・コーディングを担当しました。

デザイナーの人数が少なかったため担当範囲は広く、キャンペーン・機能追加・コンテンツ追加などサイト全体の運用を行いました。

コーディングについてはコンポーネント化を行い複数人での運用を改善しました。コンポーネント化やBEMのレギュレーションをメンバーで検討しました。

スクラム開発でアジャイル開発を行い、上場前はディレクター職が不在だったためデザイナーも施策立案を行いました。

プラン紹介ページ、料金ページ、利用事例、キャンペーン、イベントの告知などのコンテンツを制作しました。

多言語に対応しているサイトのため、UIを調整しローカライズも行いました。

料金表のUI改修

定額制のプラン変更のタイミングで料金表のデザインも見直しました。

  • 黒い帯がボタンに見えてしまっていたため、吹き出しタイプのデザインに変更
  • プランが増えたため、比較しやすいようリストでまとめて表示
  • 年間の総額を併記

応募フォーム作成

無料素材への応募を増やすため、応募フォームを作成しました。

リリースから4日で応募が1000件を超えました。

これまでメール応募だったため、システム化する事で管理も楽になりました。

改善点

  • マイページにフォームを設置した事でユーザーが募集に気づくようになった
  • 応募に必要な情報を予めフォームに引っ張ってくる事で応募の手間を減らした
  • プリクラや証明写真などの素材以外の写真が送られてくることがあったため素材番号だけ入力できるようにした

タイムズカーシェア

企業名 | パーク24株式会社

URL | https://share.timescar.jp

カーシェアサービスサイトのデザイナーとしてデザイン・コーディングを担当しました。

ブランドチェンジのタイミングだったため、共通ヘッダー・フッターのマスターブランド戦略への切り替えや、投稿キャンペーン・検定などのコンテンツ強化のワイヤーも作成しました。

アクセスログ解析

アクセスログ解析で特に成果があったのは入会フォームの最適化で、スマートフォンでの離脱率が改善しました。

Google Analyticsのデータで、入会フォームのinput画面の離脱が多かったため調査しました。

離脱の原因

  • 規約をインラインフレームで読み込んでいたが、iphoneでスクロールバーが見えていなかった
  • チェックボタンが押しづらかった
